📌 Projet Aide à la Randonnée

Université Paris Cité - Année 2024/2025 - L3P2 - jianye

Ce projet vise à améliorer l’expérience de randonnée des utilisateurs grâce à une application mobile Android qui automatise le téléchargement des mises à jour et intègre un module optionnel de reconnaissance des fleurs.

🌟 Fonctionnalités Principales

  • Automatisation du téléchargement des mises à jour pour tous les modules de l'application.

🛠️ Fonctionnalités Optionnelles et facultatives

  • Ajout d’un module de reconnaissance des fleurs via intelligence artificielle.
  • Amélioration de l’intégration avec les services de cartographie GPS.

📦 Tutoraux de ce serveur

  • 📌 Création d'un Github Pages site avec Jekyll
  • 📌 Implémentation d'un thème personnalisé

📖 Instructions d’installation

Pour installer l'application sur votre smartphone Android, suivez ces étapes :

  1. Téléchargez le fichier APK en cliquant sur le lien ci-dessus.
  2. Ouvrez les paramètres de votre téléphone et allez dans Sécurité.
  3. Activez Sources inconnues pour autoriser l'installation d'applications hors Google Play Store.
  4. Ouvrez le fichier APK téléchargé et cliquez sur Installer.
  5. Une fois l’installation terminée, ouvrez l’application et profitez de vos randonnées ! 🚶‍♂️🌿
